### CrudNode

#### Passo a passo
1. Clonar o projeto;
2. executar o comando npm install;
3. configurar o banco de dados, seja local ou remoto;
4. executar app.js

#### Rotas:
* [localhost:3000/user] - verbos a serem utilizados: POST/PUT/DELETE/GET
* [localhost:3000/swagger] Swagger - documentação da API.
